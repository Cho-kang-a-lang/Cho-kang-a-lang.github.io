# 做工仔人台語工作室 Chō-kang-á-lâng Tâi-gí Kang-chok-sek

**做工仔人台語工作室**是一个用廣播劇、有聲故事、台語詞典來守護台語ê靜態網站。

> 做寡代誌，kā咱ê記智保留--落-來

---

## 網站內容 Content

- **A郎 Podcast** — 台語中篇廣播劇
- **A郎台語小詞典** — 台語詞彙解說
- **英國囡仔古** — 英國民間故事台語版
- **民間故事** — 台灣民間故事
- **相招來看戲** — 台語劇評介紹
- **台灣舊地號名** — 台灣舊地名探索
- **台語講怪奇** — 台語奇妙知識
- **白蔥仔公主** — 台語童話故事
- **其他** — 其他台語創作

---

## 技術架構 Tech Stack

純靜態網站，無需後端框架：

- HTML / CSS / JavaScript（純前端）
- [marked.js](https://marked.js.org/) — Markdown 解析
- [Tailwind CSS](https://tailwindcss.com/) — 樣式框架
- GitHub Pages — 免費靜態網站托管
